# albumify

Finds your most listened to album based on the songs in your playlists

# create a virtual environment

```
$ python3 -m venv name
$ source name/bin/activate
```

# install dependancies

```
$ pip install -r requirements.txt
```

# for .env

```
$ pip install python-dotenv
```

# to run

```
$ python3 main.py
```
